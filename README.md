## 📌 Overview  

**Smart Public Service CRM (PS-CRM)** is a centralized digital platform designed to streamline citizen complaint management.  
It enables citizens to report issues and allows government authorities to track, manage, and resolve complaints efficiently using AI.

---

## 🎯 Problem Statement  

Citizens face daily public service issues such as:
- Poor road conditions  
- Water supply problems  
- Electricity failures  
- Garbage collection issues  

### 🚨 Current Challenges  
- Complaints are lost or delayed  
- No proper tracking system  
- Slow response from departments  
- Lack of centralized system  
- Poor transparency and accountability  

👉 Result: Inefficient governance and frustrated citizens  

---

## 💡 Proposed Solution  

PS-CRM provides a **centralized complaint management system** where:

- Citizens can submit complaints (text, image, voice)  
- AI automatically classifies and assigns complaints  
- Authorities track and resolve issues in real time  
- Citizens get live updates on complaint status

---

## 🧠 System Architecture  


Citizen → Frontend → Backend API → AI Processing → Database → Government Department → Resolution → Notification


---

## 🔄 Workflow  

1. User submits complaint  
2. AI processes and classifies issue  
3. Data stored and checked for duplicates  
4. Complaint routed to correct department  
5. Officer assigns task  
6. Field worker resolves issue  
7. Citizen receives updates
